---
title: Quick Start
description: Local development, writing workflow, and deployment
---

## 1. Run locally

```bash
pnpm install
pnpm dev
```

Open:

- `http://localhost:3000/zh/docs`
- `http://localhost:3000/en/docs`

## 2. Write docs

Keep Chinese docs in `content/docs/zh` and English docs in `content/docs/en`.

When adding pages, update each locale's `meta.json` to control sidebar order.

## 3. Build and deploy

```bash
pnpm lint
pnpm build
```

The static output is generated to `out/`, ready for Cloudflare Pages.
