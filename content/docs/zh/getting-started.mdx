---
title: 快速开始
description: 本地启动、写作与发布流程
---

## 1. 本地运行

```bash
pnpm install
pnpm dev
```

启动后可访问：

- `http://localhost:3000/zh/docs`
- `http://localhost:3000/en/docs`

## 2. 新增文档

中文文档放在 `content/docs/zh`，英文文档放在 `content/docs/en`。

如果你新增页面，也需要更新对应语言目录中的 `meta.json` 以控制侧边栏顺序。

## 3. 构建与部署

```bash
pnpm lint
pnpm build
```

构建产物在 `out/`，可直接部署到 Cloudflare Pages。
